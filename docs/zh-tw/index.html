Westwood/Santa Monica - Downtown LA
via Wilshire Bl

新的時刻表
更新： 從Westwood的Wilshire Bl以南之Veteran Av開始的所有東向Line 20班次，都將修改為行經Veteran Av、Santa Monica Bl右線、Sepulveda Bl右線、Wilshire Bl右線和前往downtown LA的一般路線。西向班次沒有變更。在Santa Monica，所有西向班次都將使用7th St，而非Lincoln Bl。
原因： 巴士將不再允許行經聯邦總務署 (General Services Administration, GSA) 大樓場地。使用Santa Monica的7th St將避免因為Lincoln Bl工程而發生的延誤。
